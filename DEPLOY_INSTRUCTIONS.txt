==========================================

STEP 1: GITHUB PUSH
-------------------
1. Open PowerShell in project folder
2. Run these commands:

   git init
   git add .
   git commit -m "Vishnu AI 1.0 - Production Ready"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/vishnu-ai.git
   git push -u origin main

   (Replace YOUR_USERNAME with your GitHub username)


STEP 2: VERCEL DEPLOYMENT
--------------------------
1. Go to: https://vercel.com
2. Sign in with GitHub account
3. Click "Add New Project"
4. Select your repository: "vishnu-ai"
5. Vercel will auto-detect Flask
6. Click "Deploy" (no build settings needed!)

IMPORTANT NOTES:
----------------
- Database uses /tmp on Vercel (ephemeral - resets on deploy)
- For production, consider using a cloud database
- Static files (CSS/JS/images) are served automatically
- All routes work out of the box


TROUBLESHOOTING:
----------------
If you see 404:
- Check vercel.json exists in root
- Check app.py is in root
- Check build logs in Vercel dashboard

If database errors:
- This is normal - SQLite on Vercel is temporary
- Use cloud DB for persistent storage
